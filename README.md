# BERT Document Classifier

A high-performance NLP pipeline for automated document classification using the BERT (Bidirectional Encoder Representations from Transformers) architecture.

## 🎯 Overview
This project aims to provide a robust, production-ready solution for categorizing documents with high accuracy. Originally developed for the Smart India Hackathon 2023, this implementation focuses on optimized preprocessing and scalable deployment.

## 🚀 Key Features
- **Context-Aware Classification**: Leverages BERT's bidirectional understanding of language.
- **Optimized Pipeline**: 40% faster preprocessing through batching and efficient tokenization.
- **API-First**: Built with FastAPI for high-performance inference.
- **Containerized**: Fully Dockerized for seamless deployment on AWS EC2 or other cloud providers.
- **Live Documentation**: [Hosted on GitHub Pages](https://Abhinav-143x.github.io/BERT-Document-Classifier/)

## 🛠️ Tech Stack
- **Language**: Python 3.11+
- **Model**: HuggingFace Transformers (BERT)
- **API**: FastAPI
- **Inference**: PyTorch
- **DevOps**: Docker, AWS EC2
